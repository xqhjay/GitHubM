## 🚀 快速开始

<a href="https://api.iconify.design/mdi:nodejs.svg?color=%23539e43" target="_blank"><img src="https://api.iconify.design/mdi:nodejs.svg?color=%23539e43" width="18" /></a> **环境要求：Node.js ≥ 20 · pnpm ≥ 9**

```bash
# 克隆
git clone https://github.com/qq5855144/GitHubM.git && cd GitHubM

# 安装依赖
pnpm install

# 启动开发
pnpm exec vite

# 构建生产
pnpm exec vite build     # → dist/
```

<div align="center">

```

                           └── Gradle APK ────► 📱 Release
```
